## Installation

1. Change into the installed directory:     
`cd 'complex plotter'`     
2. Install the required python librarys (if you already have them installed, skip this step):     
`pip install numpy `        
`pip install matplotlib`

3. Run 'install.sh':     
`./install.sh`
